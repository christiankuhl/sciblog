This is an interactive visualisation of the N-vortex problem of planar hydrodynamics in the unit disk.

The coloured dots represent vortices in the fluid, whose strengths you can change on the right. You can select one of the examples, drag vortices around, place additional vortices with a double click and delete existing vortices with a right click.

This demo is the second in a series of mathematically spirited Rust programs compiled to WASM that I'd like to showcase here. The source code for this applet, should you be so inclined, can be found here. Enjoy!
